### 数据类型

1. 整数

- 可以处理任意大小的整数，包括负整数
- 数字很大的情况下可以使用\_(下划线)分割。如 1_0_0 和 100 是一样的

2. 小数 IEEE754

- 小数也叫浮点数（因为小数点的位置可变）。

3. 字符串

- 以单引号，双引号包括的任意文本
- 字符串内部的引号，可以使用 \ 转义
- 编码
计算机只能处理数字，如果要处理文本就必须先把文本转换成数字才能处理。最早的计算机设计8个比特（bit）作为一个字节（byte）。所以一个字节能表示的最大整数是255
由于计算机最早是美国人发明的，最早只有127个字符被编入计算机，大小写字母，数字，一些特殊字符，这个编码表被称为ASCII
但要处理一个中文显然一个字节不够，至少需要两个字节，而且还不能与ASCII编码冲突，所以中国制定了GB2312编码，把中文编进去。
但是世界上的各个语言都有自己的标准，不可避免的会出现冲突，所以出现了unicode字符集（把所有语言统一到一套编码里）
统一编码之后不会有乱码问题，但是产生了另外一个问题，存储和传输的浪费。于是产生了UTF-8(可变长编码)
